Accessibility

Built-in A11y Features:

  • role="status" for screen reader announcement
  • aria-label with customizable text
  • Reduced motion support - slower/static animation
  • Dots use aria-hidden on individual dots

Usage

CSS Classes

<!-- Spinner -->
<span class="spinner spinner-md spinner-accent"></span>

<!-- Dots -->
<span class="dots dots-md dots-accent">
  <span class="dots-dot"></span>
  <span class="dots-dot"></span>
  <span class="dots-dot"></span>
</span>

<!-- Loading overlay -->
<div class="is-loading">
  <div>Content...</div>
  <div class="loading-overlay">
    <span class="spinner spinner-lg"></span>
  </div>
</div>

React Components

import { Spinner, Dots } from '@opencanoetiming/timing-design-system/react';

// Spinner
<Spinner />
<Spinner size="lg" variant="success" />
<Spinner size="sm" variant="white" label="Saving..." />

// Dots
<Dots />
<Dots size="sm" accent />
